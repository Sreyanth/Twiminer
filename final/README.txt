HOW TO RUN THIS CODE?
=============================================================
1. Run test.py to start with the classification
2. Once done, results can be seen in `tmp/results/`
3. continuous.txt is NOT being used in this final run.
4. If you want to see the extracted-feature statistics, run stats.py and open the csv in `tmp/`

MORE DETAILS
====================================
Datasets, both training and test are in the `datasets` folder
The final model, CustomizedBayes, as we call it, is in the `models/CustomizedBayes/` folder
All the generated files go to the `tmp` folder
